============================
Processes 
============================

This where we are going to take a look at the process of running a software project, and see how the roles and tools work together.

The steps include:

- Prepare the Vision Statement
- Prepare the Roadmap
- Populate the Backlog with user stories
- Break the stories into tasks
- Let the developers pull tasks and complete them
- Fulfil the testing plan
- Write/update the documentation
- Deploy
- Get feedback
- Rinse and repeat
